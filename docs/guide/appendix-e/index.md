# 附錄 E：DGX Spark 硬體規格速查表

本附錄提供 NVIDIA DGX Spark 的完整硬體規格、效能數據、比較分析與選購建議，方便讀者快速查詢與參考。

---

## 核心規格總覽

| 項目 | 規格 | 備註 |
|------|------|------|
| 晶片 | NVIDIA GB10 Grace Blackwell | 系統單晶片（SoC）設計 |
| CPU | 20 核 ARM Neoverse V2 | 支援 SMT，共 20 執行緒 |
| CPU 快取 | L2: 20 MB / L3: 共享 | 低延遲設計 |
| GPU | Blackwell 架構 | 5 世代 Tensor Core |
| Tensor Core | 第 5 代 | 支援 FP4/FP8/BF16/FP16 |
| 統一記憶體 | 128 GB LPDDR5x | CPU/GPU 共享記憶體架構 |
| 記憶體頻寬 | ~400 GB/s | 高頻寬設計 |
| 製程 | 4nm TSMC | 先進製程技術 |
| 運算能力（FP4） | 1 PFLOP | 1000 TFLOPS |
| 運算能力（FP8） | 0.5 PFLOP | 500 TFLOPS |
| 運算能力（FP16/BF16） | 0.25 PFLOP | 250 TFLOPS |

| 精度 | 算力 | 適用場景 | 備註 |
|------|------|---------|------|
| FP4 | 1 PFLOP (1000 TFLOPS) | 極致量化推論 | 最新精度，適合 LLM 推論 |
| FP8 | 0.5 PFLOP (500 TFLOPS) | 高效能推論與訓練 | Transformer Engine 支援 |
| BF16 | 0.25 PFLOP (250 TFLOPS) | 模型訓練、推論 | 動態範圍佳 |
| FP16 | 0.25 PFLOP (250 TFLOPS) | 模型推論 | 傳統 AI 精度 |
| FP32 | ~0.125 PFLOP (125 TFLOPS) | 科學計算、通用 GPU | 單精度浮點數 |
| INT8 | 2000 TOPS | 量化推論、影像處理 | 邊緣 AI 常用 |
| INT4 | 4000 TOPS | 極致量化推論 | 記憶體用量減半 |

### 記憶體架構

| 項目 | 數值 | 說明 |
|------|------|------|
| 記憶體類型 | LPDDR5x | 低功耗高效能 |
| 記憶體容量 | 128 GB | CPU/GPU 統一記憶體 |
| 記憶體頻寬 | ~400 GB/s | 高頻寬設計 |
| 記憶體位元寬度 | 256-bit | 高效資料傳輸 |
| 記憶體時脈 | ~8533 MHz | 高速時脈 |

### 統一記憶體優勢

| 傳統架構 | DGX Spark 統一記憶體架構 |
|---------|------------------------|
| CPU 與 GPU 各有獨立記憶體 | CPU 與 GPU 共享 128 GB 記憶體 |
| 需要透過 PCIe 複製資料 | 零複製，直接存取 |
| 資料傳輸延遲高 | 延遲極低 |
| 記憶體容量受限於較小的一方 | 128 GB 全量可用 |
| 需要手動管理記憶體搬移 | 自動管理，程式簡化 |

## 選購建議

### 適合 DGX Spark 的使用場景

| 場景 | 適合度 | 說明 |
|------|--------|------|
| LLM 推論（7B-70B） | ★★★★★ | 128 GB 記憶體可運行大型模型 |
| LLM 微調（LoRA/QLoRA） | ★★★★★ | 記憶體充足，效能優異 |
| 影像生成（FLUX/SDXL） | ★★★★★ | GPU 效能強大 |
| RAG 系統部署 | ★★★★★ | 可同時運行嵌入模型 + LLM |
| AI Agent 開發 | ★★★★★ | 多模型同時運行 |
| 科學計算（RAPIDS） | ★★★★ | GPU 加速資料處理 |
| 大型模型從頭訓練 | ★★★ | 受限於單機記憶體 |
| 遊戲 | ★★ | 非設計用途 |
| 一般文書處理 | ★★ | 殺雞用牛刀 |

### 不適合的場景

| 場景 | 原因 | 建議替代方案 |
|------|------|-------------|
| 大型模型從頭預訓練 | 記憶體與算力限制 | DGX H200 / DGX B200 |
| 高效能遊戲 | 驅動與生態系差異 | RTX 5090 / 4090 |
| 超大型叢集訓練 | 單機限制 | DGX SuperPOD |
| 預算有限的個人專案 | 價格較高 | RTX 4060 Ti 16GB / Mac M4 |
